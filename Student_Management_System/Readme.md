# Student Management System (Python CLI)

A modular Python CLI application to manage student records using JSON storage.

## Features
- Add student
- View students
- Search student
- Update marks
- Delete student
- Persistent JSON storage
- Modular OOP design

## Project Structure
student_manager/
cli/
data/

## How to Run
From project root:

python -m cli.main

## Concepts Practiced
- OOP
- File handling
- JSON storage
- Modular architecture
- Dependency injection
